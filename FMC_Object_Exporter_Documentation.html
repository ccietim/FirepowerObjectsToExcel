üìã Description

This Python script connects to Cisco Firepower Management Center (FMC) via the fireREST API, exports host, network, and network group objects, and compiles a clean Excel spreadsheet (FMC_Object_Export.xlsx) for easy analysis.

  • Processes host and network entries (value, name, type)
  • Processes networkgroup entries, excluding:
    • Groups containing only FQDN entries
    • Groups containing 0.0.0.0/0 or ::/0 literals
  • Resolves object references in groups using host.json

üöÄ How to Run

  1. Enable REST API access in FMC: System > Configuration > REST API Preferences
  2. Run the script: python EXPORT.py
  3. Enter FMC IP/hostname, username, and password when prompted
  4. The script generates:
    • host.json, network.json, networkgroup.json
    • FMC_Object_Export.xlsx
